Belajar Console

  • Javascript memiliki fitur untuk melakukan logging yaitu bernama console
  • Loggin merupakan mekanisme untuk menampilkan informasi yang biasa dilakukan oleh programmer, tanpa harus menganggu alur dari aplikasi dan juga interaksi dengan user
  • Cara menggunakannnya yaitu console

Console Method

Jadi biasanya ada method yang digunakan setelah penulisan console.(..)

Console Method Keterangan
console.info(..) / console.log(..) Untuk memberitahu informasi
console.warn(..) Untuk memberikan peringatan
console.error(..) Untuk memberikan pesan error
console.table(..) Untuk menampilkan pesan dalam bentuk table
