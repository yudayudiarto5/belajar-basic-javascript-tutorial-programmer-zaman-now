Belajar Do While Loop

Do while loop hampir mirip dengan kondisi while loop, yang menjadi perberdaannya yaitu do while loop akan mengeksekusi block perulangan minimal 1x, kenapa? Karena do while loop akan melakukan pengecekan kondisi setelah block perulangan walaupun nilai awalnya false.

Sintaks Perulangan While

                init statement; 
                do {
                    // block perulangan
                    // post statement
                }
                while(kondisi);

Keterangan

  • Init statement hanya akan di eksekusi 1x saja di awal perulangan
  • Kondisi akan selalu dilakukan perulangan jika kondisinya true, dan akan berhenti jika kondisi false
  • Post statement akan selalu dieksekusi di akhir statement
  • Init statement, kondisi, dan post statement tidak wajib diisi, namun nanti hasilnya akan true sehingga menyebabkan perulangan tanpa henti!
