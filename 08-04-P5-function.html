Belajar Function / Method

  • Function akan berjalan saat kita memanggilnya
  • Untuk membuat function, kita bisa menggunakan kata kunci function lalu diikuti dengan nama function, kurung () dan di akhiri dengan kurung {}
  • Cara memanggil function yaitu nama function diikuti kurung ()
  • Function bisa dipanggil berulang kali
