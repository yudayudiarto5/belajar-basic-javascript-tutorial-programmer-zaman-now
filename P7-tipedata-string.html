Escape Sequence

JavaScript mendukung escape sequence di string. Esacape sequence merupakan karakter khusus, seperti Enter, Tab, " (kutip dua). dan lain-lain. Berikut contoh escape sequence yang didukung oleh JavaScript di data string.

Karakter (Escape) Keterangan
\n ENTER
\t TAB
\' '
\" "
\\ \
