Belajar Operasi Logika

  • Operasi logika merupakan operasi yang digunakan untuk 2 (dua) buah data boolean
  • Hasil yang didapatkan dari operasi logika yaitu boolean (true/false)

Operator Perbandingan

Operator Keterangan
&& Dan
|| Atau
! Kebalikan

Operator && (dan)

Nilai 1 Operator Nilai 2 Hasil
true && true true
true && false false
false && true false
false && false false

Operator || (atau)

Nilai 1 Operator Nilai 2 Hasil
true || true true
true || false true
false || true true
false || false false
