Belajar Tipe Data Boolean

  • - Tipe data boolean merupkan tipe data yang berisikan data kebenaran, artinya hanya ada dua data (kondisi) bisa benar atau salah.
  • - Benar di presentasikan true, sedangkan salah itu false
  • - Penggunaan tipe data boolean ini sangat powerfull dalam bahasa pemograman.
