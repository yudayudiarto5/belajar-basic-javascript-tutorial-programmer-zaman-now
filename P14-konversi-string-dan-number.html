Belajar Konversi String dan Number

  • Saat membuat aplikasi, kadang input dari pengguna selalu dalam bentuk string
  • Sedangkan kita ingin mengelola datanya dalam bentuk Number
  • Maka sangat disarankan untuk melakukan konversi tipe data
Function (fungsi) Keterangan
parseInt(string) Mengubah String ke Integer
parseFloat(string) Mengubah String ke Float (Floating Point)
Number(string) Mengubah String ke Number (hasilnya bisa berupa bilangan bulat maupun bilangan decimal)
number.toString() Mengubah number ke string

NaN (Not a Number)

  • Jika data string yang kita konversi ke number bukan berupa data yang valid
  • Maka data yang tidak valid tersebut menghasilkan NaN (Not a Number0
  • NaN merupakan Number spesial yang menyebutkan bahwa ini bukan Number
  • Selain itu, Data NaN jika dioperasikan dengan lainnya, tetap akan menjadi NaN
  • Fungsi parseInt dan parseFloat akan mentolerir jika data yang didepannya memang berupa angka, contohnya "3xx" maka hasilnya 3. Namun fungsi Number() akan tetap tidak mentolerir

Fungsi isNan

Fungsi IsNan digunakan untuk mengecek apakah data tersebut merupakan NaN atau Bukan, fungsi isNan akan menghasilkan true atau false
