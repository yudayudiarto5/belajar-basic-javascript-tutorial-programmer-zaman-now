Belajar Typeof

Tyopeof merupakan operator yang mempunyai fungsi untuk melihat tipe data yang digunakan pada sebuah variable. Karena Javascript merupakan dynamic language maka kita perlu untuk mengetahui tipe data yang digunakan dan akan menghasilkan sebuah string.

Type dan hasil yang typeof pada javascript

Type Hasil TypeOf
Undefined "Undefined"
Null "Null"
Boolean "Boolean"
Number "Number"
BitInt "BitInt"
String "String"
Symbol "Symbol"
Function "Function"
Lainnya, seperti array, null "Object"
