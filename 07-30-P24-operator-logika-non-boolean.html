Belajar Operator logika di Non Boolean

Sebelumnya kita ketahui bahwa penggunaan logika biasa digunakan pada data boolean. Namun pada Javascript operator logika bisa digunakan pada data yang Non Boolean

Operator || (OR)

  • Operator logika || (OR) membaca data dari kiri ke kanan
  • Operator ini akan mengambil data truthy yang pertama
  • Namun jika kedua data nya falshy maka akan mengambil data falshy yang paling kanan.

    • Operator && (AND)

  • Operator logika && membacata data dari kiri ke kanan
  • Operator ini akan mengambil data falshy yang pertama
  • Namun jika kedua data nya truthy maka akan mengambil data truthy yang kedua.
