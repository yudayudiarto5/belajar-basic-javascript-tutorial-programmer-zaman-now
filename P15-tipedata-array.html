Belajar Tipe Data Array

  • Array merupakan tipe data yang isinya kumpulan
  • Array di JavaScript bersifat dinamis, artinya data bisa bertambah dengan sendirinyaa saat kita memasukan data array baru.

Cara Kerja Array

  • Setiap array akan disimpan dalam posisi berurutan, dimana urutan pertama dimulai dari 0
  • Setiap kita menambahkan data array baru, maka akan disimpan di urutan terakhir
  • Urutan Array, disebut dengan index
Operasi / Function Keterangan
namaArray.push(value)( Menambahkan data ke Array, boleh lebih dari 1 sekaligus
namaArray.length Untuk mendapatkan panjang Array
namaArray[index] Mendapatkan data di posisi Index
namaArray[index] = value Mengubah atau Mengupdate data di array
delete namaArray[index] Menghapus data di array, namun tidak menghapus index (jadi tidak bergeser)
